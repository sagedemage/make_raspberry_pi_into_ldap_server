# Make a Raspberry Pi Into a LDAP Server
An article to make a Raspberry Pi into a LDAP server.

## Setup

Install all dependencies
```
pnpm install
```

Start the website
```
pnpm dev
```

Build the website
```
pnpm build
```

Preview the built website
```
pnpm preview
```

## Why use pnpm?
1. Save disk space
2. Boost installation speed
